# Düsseldorf Sehenswürdigkeiten

Eine interaktive, elegante HTML-Seite die drei der bekanntesten Sehenswürdigkeiten Düsseldorfs vorstellt.

---

## Inhalt

Die Seite präsentiert folgende Sehenswürdigkeiten:

- **Rheinturm** – Fernsehturm mit Aussichtsplattform und Dezimaluhr
- **Altstadt** – Die längste Theke der Welt mit Geschichte und Kultur
- **Kunstpalast** – Eines der bedeutendsten Kunstmuseen in NRW

Jede Sehenswürdigkeit enthält:
- Beschreibung mit Geschichte und Besonderheiten
- Adresse und Öffnungszeiten
- Eintrittspreise (falls vorhanden)

---

## Vorschau

Drei Karten nebeneinander, jede mit eigener Akzentfarbe. Hover-Effekte und sanfte Ladeanimationen sorgen für ein modernes Nutzererlebnis.

---

## Verwendung

Keine Installation nötig. Einfach die Datei im Browser öffnen:

```
index.html → Doppelklick oder per Browser öffnen
```

Funktioniert in allen modernen Browsern (Chrome, Firefox, Safari, Edge).
